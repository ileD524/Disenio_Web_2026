Estilización de texto en CSS

En este ejercicio, estamos experimentando con diferentes propiedades de texto para mejorar la legibilidad y el estilo de nuestro contenido. Podemos cambiar la familia de fuentes, el tamaño de fuente, el peso de la fuente, el color y la alineación del texto para crear un diseño atractivo y fácil de leer. La clave es encontrar un equilibrio entre estilo y legibilidad para que el contenido sea agradable a la vista y fácil de entender.
